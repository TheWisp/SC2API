bool SC2API::Unit::IsOwnedByEnemyPlayer ( ) const

Determines whether unit is owned by an enemy of local player. Note it would return true for an enemy unit in fog of war, despite not knowing its existence.

Returns
True when the unit is known to not owned by any friendly or neutral player. False when the unit is owned by friendly or neutral player, or was confirmed dead.
